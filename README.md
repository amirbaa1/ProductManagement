# پروژه مدیریت انبار و فروشگاهی

## Running the site server
برای اجرای سایت ادرس زیر در ترمینال خود اجرا کنید 
```python
python manage.py runserver
```
و ادرس لینک `http://127.0.0.1:8000/`وصل شوید.
